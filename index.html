Da Macroeconomia ao Impacto Real

Como graduanda em Economia, a minha principal motivação é compreender a mecânica dos mercados. O meu foco é utilizar Python, análise de dados e BI para modelar cenários. Seja para antecipar movimentos de mercado, otimizar cadeias de abastecimento ou estruturar análises financeiras, o meu objetivo é fornecer a base analítica necessária para a tomada de decisão estratégica em qualquer área de negócio.

Estudo de Caso

Automação e Análise do Boletim Focus

Como estruturei um pipeline de dados em Python para antecipar movimentos de juros e inflação, auxiliando na mitigação de risco.

Em breve: Análise Visual do Focus

O gráfico com os resultados da extração será publicado em breve.

O Problema

A coleta manual de indicadores do Banco Central consumia tempo e abria margem para erros humanos na projeção de custos.

A Solução

Script em Python que consulta a API do SGS (Bacen), limpa os dados, estrutura num DataFrame e plota as tendências automaticamente.

O Impacto

Previsibilidade para a tomada de decisão corporativa, permitindo negociações de longo prazo antes de altas da inflação.
